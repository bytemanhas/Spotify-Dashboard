# Spotify Analytics Dashboard | Power BI Project 🎵

## Overview

This project is an end-to-end Spotify Analytics Dashboard developed in Power BI to analyze music streaming performance, artist trends, genre dominance, label market share, and audio intelligence.

The dashboard transforms raw music-related data into actionable business insights using interactive visualizations, DAX measures, and data modeling.

---

## Dashboard Pages

### 1. Executive Overview

* Total Streams
* Total Tracks
* Total Artists
* Total Albums
* Streaming Trend Analysis
* Global Stream Distribution
* Genre Performance

### 2. Artist Analytics

* Top Performing Artist
* Top Genre
* Top Label
* Artist Stream Contribution
* Genre Market Share
* Label Market Share

### 3. Audio Intelligence Dashboard

* Average Danceability
* Average Energy
* Average Tempo
* Average Loudness
* Popularity vs Danceability Analysis
* Explicit vs Clean Content
* Major vs Minor Mode Analysis

## Key Insights

* Hip-Hop emerged as the highest-streaming genre.
* Taylor Swift ranked as the top-performing artist.
* Republic Records led label market share.
* Higher danceability tracks showed stronger popularity trends.
* Audio features like energy and tempo significantly influenced listener engagement.
